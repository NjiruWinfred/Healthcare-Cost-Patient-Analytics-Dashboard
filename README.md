# Healthcare-Cost-Patient-Analytics-Dashboard
**Project Overview**
This project analyzes healthcare records to identify financial, demographic, and operational trends using Microsoft Excel.
**Objectives**
Clean messy healthcare data
Analyze treatment costs
Identify patient demographic patterns
Build interactive PivotTables and charts
Generate business insights
**Skills Demonstrated**
Excel Data Cleaning
VLOOKUP
Pivot Tables
Conditional Formatting
Data Visualization
Financial Analysis
Healthcare Analytics
**Key Insights**
Diabetes treatments generated the lowest average costs, while flu generated the highest average costs. 
Patients between 15 and 64 years recorded the most diagnoses. 
Female patients represented the majority of visits. 
Certain doctors managed significantly higher treatment costs than others. 
**Some Key Findings**
Chronic conditions such as diabetes contributed significantly to treatment costs.
Older patients showed higher average healthcare expenditure
Data quality issues highlighted the importance of proper healthcare reporting systems
